Welkom op de motivatie website van Nick Geense.

Beste bezoeker, je bent hier aanbeland op een website die geen ander doel dient dan mijn carrière in de IT kracht bij te zetten. Dit ga ik doen door te laten zien dat ik affiniteit heb met IT, mede door mijn studie, maar ook door mijn hobby activiteiten.

De afgelopen 2,5 jaar ben ik steeds meer bezig met het programmeren van software op allerlei vlakken. Voor mijn werk heb ik kleinere applicatie moeten schrijven voor het automatiseren van veel voorkomende handelingen en ik heb een simpel systeem opgezet voor projectmanagement.

Going the extra mile

Ik wil niet zomaar in de IT werken, nee ik wil professioneel kunnen werken in de IT. Ik wil ervaring op doen met werken in teamverband door middel van Scrum en Agile. In mijn eentje voel me beperkt in de vooruitgang en twijfel soms over implementaties van code door het gebrek aan feedback. Ik heb deze website gemaakt om aan te geven dat ik het kan en omdat ik het nodig acht om mij te onderscheiden van de rest.

Deze website is gemaakt in Django omdat het verder gaat dan een wysiwig editor waar je tekst en plaatjes op knalt. Ik wilde mijzelf uitdagen door niet voor de makkelijke oplossing te kiezen. Ik wil begrijpen hoe het werkt, wat je er mee kan. En wie weet waar ik deze website nog voor ga gebruiken in de toekomst.
